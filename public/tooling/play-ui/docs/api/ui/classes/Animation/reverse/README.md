# `.reverse()`

> The `animation.reverse()` method.

This [Animation](..) instance method is used to reverse an animation.

## Syntax

```js
animation.reverse();
```

**Return**

+ **`this`** - The `Animation` instance.

## Usage

Obtain an instance of an ongoing animation and call `.reverse()`.

```js
animation.reverse();
```
