# `.pause()`

> The `animation.pause()` method.

This [Animation](..) instance method is used to pause an animation.

## Syntax

```js
animation.pause();
```

**Return**

+ **`this`** - The `Animation` instance.

## Usage

Obtain an instance of an ongoing animation and call `.pause()`.

```js
animation.pause();
```
